CWU Letter Occurrence

The purpose of this project is to find letter occurrences in a text file. After running the code, the user will be able to see the number of each letter [a-zA-Z] with uppercase and lowercase being counted the same. Additionally, a bar graph will help visualize the letter contents of the file provided.

build:

g++ CWULetterOccurrence.cpp -o CWULetterOccurrence

run with an example file (change example.txt to the file you want to analyze):

./CWULetterOccurrence example.txt

Command Line Arguments

-i ignore displaying the bar graph
